1. Avoid repetitive card grids: Do not default to 3-column card layouts for every section. Mix up your section types to create visual rhythm (e.g., alternating text-image splits, full-width highlights, asymmetric lists).
2. Differentiate the Hero: The hero section should feel bespoke, not like a generic "Title, Subtitle, 2 Buttons" template. Consider bold typography, full-bleed images, or creative layouts.
3. Authentic Copywriting: Never use "Lorem Ipsum" or generic filler text like "We provide the best services". Write realistic, brand-specific copy based on the given prompt context.
4. Purposeful Whitespace: Use generous and deliberate spacing between sections and elements. Do not cram elements together.
5. Meaningful Imagery: Avoid generic stock photos if possible. Choose specific, evocative descriptions for images.
