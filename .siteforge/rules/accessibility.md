1. Color Contrast: Text colors must have a high contrast ratio against their background color (at least 4.5:1 for normal text).
2. Semantic HTML: Use appropriate heading tags (h1, h2, h3) in sequential order. Do not skip heading levels for styling purposes.
3. Interactive Elements: Buttons and links must have clear hover and focus states for keyboard accessibility. Include `aria-label` or accessible names for icon-only buttons.
4. Images: All `<img>` tags MUST have a descriptive `alt` attribute summarizing the content. Decorative images should have `alt=""`.
5. Readable Fonts: Do not use overly thin or illegible font weights (e.g. `font-thin`) for body text. Font size should default to a readable baseline.
